## 🧪 Featured — SEM · ~50,000×

> Electrons carry no color. From here on, everything is grayscale.

### [BraggSpect](https://github.com/FrankLovelace/BraggSpect)
> *X-ray Diffraction Analysis Platform*
A high-performance analytical platform for crystallographic phase identification and structural refinement of XRD data. A hybrid microservices architecture (**.NET 8 + Python HPC**) distills the 100 GB COD database into an ultra-optimized local index, enabling real-time **Hanawalt searches** and **Rietveld refinements** with scientific-grade accuracy.

### [VAK Dashboard](https://github.com/FrankLovelace/EnlacesQuimicos) · [live](https://enlacesquimicos.franklovelace.dev)
> *Van Arkel–Ketelaar Bond Calculator*
Find the bond formed by two atoms from their symbol and electron count — chemistry running in **WebAssembly** for speed and efficiency.

### [Distributed Infrastructure ARM64](https://franklovelace.dev)
> *High Availability & Zero Trust Security*
Microservices orchestration on Oracle Cloud (Ampere): Edge / Compute / Data layers, **Security-by-Design** with Cloudflare Tunnels (Zero Ingress), Docker network micro-segmentation, Java 21 Generational ZGC for sub-millisecond GC pauses, and automated off-site disaster recovery.

### [Data Science & Predictive Modeling](https://github.com/FrankLovelace/Data-mining)
> *Full Data Mining Cycle*
ETL, feature engineering and statistical testing (Shapiro–Wilk, Kruskal–Wallis, p < 0.001), then modeling: linear regression (R² 0.66), KNN classification, and K-Means virality clustering — communicated through Seaborn visualizations.
