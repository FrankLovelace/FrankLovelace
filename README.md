## Featured

###  [Corporate B2B Platform](http://franklovelace.dev/proyectos)
> *Hybrid Architecture .NET 8 + Vue.js*
A decoupled ecosystem featuring a public-facing portal (Vue.js) and a secure administrative panel (Blazor Server). Optimized with custom **IntersectionObserver** infinite scrolling and Lazy Loading. Deployed on Azure Web Apps via GitHub Actions.

### [Distributed Infrastructure ARM64](https://franklovelace.dev/proyectos)
> *High Availability & Zero Trust Security*
Microservices orchestration on Oracle Cloud (Ampere). Features a **Security-by-Design** approach using Cloudflare Tunnels (Zero Ingress), Docker micro-segmentation, and a generational ZGC memory strategy for sub-millisecond latency.

### [Neural Audio Pipeline](https://franklovelace.dev/proyectos)
> *AI-Driven Audio Synthesis*
An ETL pipeline that uses **Meta's Demucs** and **Spotify's Basic Pitch** neural networks to isolate vocals and convert audio to MIDI/Binary formats. Implements a Master-Slave synchronization protocol for real-time broadcasting.

### [Refactoring Open Source](https://github.com/FrankLovelace/BetterRTP-1.21)
> *Software Maintenance & Migration*
Resurrected a legacy Java codebase by auditing Maven dependencies, resolving breaking API changes, and migrating the build lifecycle to Java 21 LTS.
